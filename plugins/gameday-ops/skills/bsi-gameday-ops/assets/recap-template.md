# Post-Game Recap Output Template

```
FINAL: [Away Team] [Score], [Home Team] [Score]

THE HEADLINE
→ [One sentence: the story of this game — not the score, the tension]

THREE FACTS
1. [Key stat or play that decided it]
2. [Pitching line or offensive performance]
3. [What this means for standings/rankings/postseason]

WHAT'S NEXT
→ [Next game in series, or next opponent + date]

Data via BSI as of [HH:MM CT]
```

## Rules
- Headline is the STORY, not the box score. "State's bullpen collapse" not "State loses 7-4"
- Three facts, not five. Not two. Three.
- Each fact is one sentence. Two max.
- WHAT'S NEXT is forward-looking — where does this go from here
- Total output: ≤150 words. This is a recap, not a feature.
