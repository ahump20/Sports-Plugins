# Game Focus Output Template

```
[TEAM A] vs [TEAM B] — [Score] | [Inning]
Last updated: [HH:MM CT] via [source]

SITUATION
→ [Runner positions, outs, count if available from play-by-play]
→ [Leverage: high/medium/low — one line on why]

KEY DEVELOPMENTS (since last check)
→ [What happened — specific plays, pitching changes, scoring]
→ [Second development]

GAME NARRATIVE
→ [2-3 sentences: how we got here, what's at stake, what to watch next]
```

## Rules
- Situation block comes first — it's what the user is asking about
- Key developments only include what changed since the last update
- Game narrative is context, not recap — forward-looking
- If this is the first check, "KEY DEVELOPMENTS" becomes "GAME STORY SO FAR"
