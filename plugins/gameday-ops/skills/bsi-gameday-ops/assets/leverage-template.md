# Leverage Situation Output Template

```
⚡ HIGH LEVERAGE — [Game], [Inning]

SITUATION: [Runners, outs, score, batter vs. pitcher if available]
STAKES: [What happens if Team A scores/doesn't — conference race, ranking, postseason]

CONTEXT:
→ [Relevant pattern from data — e.g., bullpen usage, platoon split]
→ [Historical base rate if applicable — flagged as training data]

WHAT TO WATCH:
→ [Specific matchup advantage or decision point]

Data via BSI as of [HH:MM CT]
```

## Rules
- Lead with the situation, not the context
- Stakes are concrete: "A win moves Texas to 2nd; a loss drops them to 5th"
- Historical patterns flagged as training data, not presented as live facts
- One "WHAT TO WATCH" item. Not three. The most important thing.
- Total output: ≤100 words. The moment is now — don't write an essay about it.
